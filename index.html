    Q&A

    What are the differences between Web 1.0, Web 2.0 and Web 3.0?

    The major difference between Web 1.0 and the rest is that Web 1.0 did not offer an interactive experience to the user. The pages were static, not allowing the user to filter information, or post feedbacks. In addition, the design of the pages was basic, and data was updated only every once in a while. As a result, the web consisted of mostly personal pages.
    Web 2.0 changed that by allowing interaction between site users and site owners. Users were able to sort information, and the content of the pages became dynamic and responsive. As a result of that different kind of websites like Facebook, Wikipedia, blogs etc were created. Another significant change was the abilty to search the web more efficiently, for example, by using Google search engine.
    Web 3.0 is about altering the Web into a database, and making it more semantic. It uses artificial intelligence in order to provide faster and more relevant results. One example is the result of a search query: retrive the actual answer to a query, instead of a list of websites that are relevant to it. In addition, graphics are more rich, including 3D graphics.
