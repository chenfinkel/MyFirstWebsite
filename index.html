    Q&A

    What are the differences between Web 1.0, Web 2.0 and Web 3.0?

    1. User interaction - The major difference between Web 1.0 and the other two is that Web 1.0 did not offer an interactive experience to the user. The pages were static, not allowing the user to filter information or post feedbacks, for example. Web 2.0 and Web 3.0 changed that by allowing interaction between site users and site owners.
    2. Types of pages - Another difference is the types of pages the web consisted of. Web 1.0, as a result of being non-intractive, consisted of mostly personal pages, as opposed to Web 2.0 and Web 3.0 that consists of sites like blogs, social media, Wikis etc.
    3. Search - In Web 1.0 search is very fast but many times results are inaccurate. In Web 2.0 search engines improved and are able to retrive a lot more relevant sites, and more accurate results. In Web 3.0 search engine will be able to retrive the actual answer to a query, and not a list of websites, using the data on the web.




    An example that demonstrates the differences:

    Searching a query: "best rock bands in 1999"

    Web 1.0 - returns many unrelevant websites. For example - sites containing informations about rocks (stones). Also, sites that could be relevent but do not conatin the exact query terms will not be retrived (For example, sites about "most popular punk ensembles"). It will be hard to find the sites I wanted in the results.
    Web 2.0 - returns a list of relevant websites: Lists of best rock groups in 1999, best bands in subgenres of rock, sites about popular bands in 1999 and many more. Most of the retrived sites will be informative for me, and probably will not be missing any important site.
    Web 3.0 - returns a list of the best rock bands in 1999, using the information found on the web.
